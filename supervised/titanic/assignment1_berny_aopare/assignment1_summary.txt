1. PROBLEM STATEMENT


The problem being solved is predicting the survival of passengers aboard the 
Titanic based on various features such as age, gender, passenger class, fare, 
and family relationships.

This is a classic machine learning problem where we analyze historical data 
from the Titanic disaster to build a predictive model that can determine 
whether a passenger survived or not based on their characteristics.



2. PROBLEM CATEGORIZATION


Type of Learning: SUPERVISED LEARNING
- Reason: We have labeled training data where the outcome (Survived: 0 or 1) 
  is known for each passenger. The model learns from these labeled examples.

Type of Task: BINARY CLASSIFICATION
- Reason: We are predicting two distinct classes - whether a passenger survived 
  (1) or died (0). This is a classification problem, not regression, as we're 
  predicting discrete categories rather than continuous values.

Learning Style: BATCH LEARNING
- Reason: The model is trained on a complete, fixed dataset all at once. The 
  training data doesn't change over time, and we're not updating the model 
  incrementally with new data (which would be online learning).

4. MODEL TRAINING AND RESULTS


Model Selected: Random Forest Classifier
- Algorithm: Ensemble learning method using 100 decision trees
- Max depth: 5 levels to prevent overfitting
- Random state: 42 (for reproducibility)

Features Used for Training:
1. Pclass (Passenger Class)
2. Sex (Gender - encoded)
3. Age (filled missing with median)
4. SibSp (Siblings/Spouses)
5. Parch (Parents/Children)
6. Fare
7. Embarked (Port of embarkation - encoded)
8. FamilySize (Engineered feature: SibSp + Parch + 1)

Data Preprocessing:
- Missing Age values filled with median (29.7 years)
- Missing Embarked values filled with mode (S - Southampton)
- Categorical variables (Sex, Embarked) encoded using Label Encoding
- Feature engineering: Created FamilySize feature

Train-Test Split:
- Training set: 80% (712 passengers)
- Testing set: 20% (179 passengers)
- Split method: Random with stratification
